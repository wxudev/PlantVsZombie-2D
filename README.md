# 植物大战僵尸（2D复刻版）

基于Unity引擎开发的2D塔防游戏，复刻植物大战僵尸核心玩法。

## 项目介绍
玩家通过种植向日葵和豌豆射手，抵御僵尸的进攻，包含完整的阳光经济系统、植物放置系统、僵尸战斗逻辑。

## 技术栈
- Unity 2D
- C#
- DOTween动画插件
- 对象池优化
- 状态机模式

## 核心功能
-  向日葵定时产阳光 + 天空随机掉落阳光双来源经济系统
-  卡片选择种植，冷却CD、阳光不足、可种植三态状态机
-  豌豆射手自动攻击，使用对象池优化子弹性能
-  僵尸行进、啃食植物、受伤掉头颅、死亡等完整AI逻辑
-  2D物理碰撞检测，豌豆子弹伤害判定
